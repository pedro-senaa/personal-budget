# WHAT IS THIS?

back-end for and [envelope-like budgeting method](https://www.thebalancemoney.com/what-is-envelope-budgeting-1293682) . Has base functionalities (GET, POST, DELETE, PUT) and a **transfer** method

for the transfer method, url must be `/transfer/:from/:to` with proper id's, and body to have `amount` key set to a number.

# HOW TO USE

Must have express installed. Then, just run server.js and hf :)
